Reflectcal mailer
================

Daemon for sending e-mail, part of reflectcal.com.

Installation and run
--------------------

1. Clone this repository locally.

  ```
  mkdir reflectcal
  cd reflectcal
  git clone https://github.com/alexeykomov/reflectcal.git
  ```

8. Go to [http://localhost:3000/](http://localhost:3000/)
